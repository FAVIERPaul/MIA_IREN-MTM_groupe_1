Q: Comment lancer l'app?
A: npm start → puis http://localhost:5001

Q: Comment modifier les données?
A: Ouvrez data.js → modifiez mockUsersData → rechargez la page

Q: Comment ajouter un utilisateur?
A: Voir EXAMPLES.js → copier-coller → modifier avec vos données

Q: Le port 5001 est occupé?
A: Changez le port dans server.js (ligne 3: const PORT = 5002)

Q: Les dépendances ne s'installent pas?
A: Installez Node.js: conda install -c conda-forge nodejs -y

Q: Plus de détails?
A: Lisez README.md ou INDEX.md
